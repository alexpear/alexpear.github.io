Fish Tank
This is a prototype of a website.
The site will display a window into a little world that runs in real time for the viewer's interest.


Prerequisites:
(Time of writing 2019 April 19)
the fishTank/ directory and its package.json file
node.js installed
npm installed
sudo npm i http-server -g
a browser

How to run locally:
go to the fishTank directory in the terminal
npm start
go to localhost:8080 in the browser

(see package.json's scripts section for backstage details)
